== Description ==

> #### Major features in HandL YouTube Extra includes:
>
> * Embed multiple YouTube videos on your lead pages (fully trackable individually)
> * Get real-time statistics (email, seconds watched, where (seconds) user left, percentage of watched w/ heatmap support)
> * Remembers/resumes where user left off on a video. 

[Check out the documentation](http://www.haktansuren.com/handl-youtube-extra/?utm_medium=referral&utm_source=wordpress.org&utm_campaign=HandL+YouTube+Extra+Readme&utm_content=Documentation)

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `handl-youtube-extra` folder to the `/wp-content/plugins/` directory via FTP
1. Activate the plugin through the 'Plugins' menu in WordPress
